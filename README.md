# Chill Mole POS

Sistema de gestión para Chill Mole, tienda familiar de moles.

## Roles
- **Administrador:** gestionar pedidos, ventas, clientes, ingredientes, recetas, platillos y categorías.
- **Empleado:** ver platillos, agregarlos al carrito y realizar ventas o pedidos.
- **Cliente:** explorar el catálogo y hacer pedidos.

## Descripción
Optimiza la administración de la tienda y mejora la experiencia de los usuarios.
